发表于 2019-12-25 更新于 2019-12-05 分类于 os

Linux 我应该是努力过多次，始终只是皮毛。这次还是打算全面的了解 Linux 的运维技术和最新的技术趋势。采用的 os 版本是 centos 7.x。

目录结构和分区

我们在一个新的硬盘上安装 Linux，一般会做如下考虑：

  • 一个 /boot 区，一般 200M 左右，安装 Linux 的 Kernel 文件。单独一个区，在系统出现问题的时候会比较好解决
  • 一个 swap 分区，一般是内存的2倍以上
  • 一个 / 分区，一般 10G 以上
  • 剩下的可以给 /data，作为数据分区

对于 MBR 分区来说，一个硬盘最多只能有四个主分区或者扩展分区（扩展分区最多一个），一个扩展分区上可以有若干个逻辑分区

  • 对于 Linux 来说，一个 SATA 硬盘，前面四个分区是供主分区和扩展分区使用，分别为 /dev/sda1 ~ /dev/sda4。逻辑分区从 /dev/sda5 开始

Linux 常用简单命令集

# free -h        //以 M 为单位看系统内存
# lsblk            //系统的分区情况
# lscpu            //看 CPU 的情况
# cat /proc/meminfo            //获得和内存相关的信息
# cat /etc/centos-release   //安装的 centos 版本
# lsb_relase -a                //安装的 centos 版本
# uname -r                     //内核的版本
# tty                              //当前终端名
# who      //当前登陆本服务器所有的终端名
# whoami    //我是谁，简略信息
# who am i     //我是谁更详细的信息，包括登陆的终端名和 IP 地址
# echo $SHELL   //当前运行的 Shell
# cat /etc/shells    //当前系统支持的所有 Shell，对于/usr/sbin/nologin shell，主要是给应用程序使用。不可以登陆
# echo $PS1
# PS1="[\u@\h]$"    //去掉 working directory
# PS1="\[\e[1;5;41;33m\][\u@\h \w]\\$\[\e[0m\]"     //对于生产环境建议用特别的提示标识，5表示闪烁
# which <command>
# whereis <command>     //和该命令相关的一些文档信息
# hash           //当前被缓存的命令路径
# alias cdnet = "cd /etc/sysconfig/network-scripts/"     //别名一般保存在用户家目录中的 .bashrc 中
# unalias <别名>      //取消别名
# . .bashrc     //使修改后的 .bashrc 立即生效，等同于 source .bashrc
